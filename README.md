# 🌱 HortLink Mobile

Aplicativo mobile desenvolvido para conectar pequenos produtores rurais diretamente ao cliente final.

O projeto tem como objetivo facilitar a divulgação e comercialização de produtos agrícolas locais, oferecendo uma plataforma simples, acessível e intuitiva.

---

# 📱 Sobre o Projeto

O HortLink Mobile foi desenvolvido como projeto acadêmico na área de tecnologia e impacto social.

A proposta do aplicativo é aproximar:

* 👨‍🌾 Pequenos produtores
* 🛒 Consumidores finais

permitindo que produtores publiquem seus produtos agrícolas e que clientes possam visualizar os itens disponíveis diretamente pelo aplicativo.

---

# 🚀 Funcionalidades

## 🔐 Autenticação

* Cadastro de usuários
* Login com Firebase Authentication
* Persistência de sessão

## 🥬 Produtos

* Cadastro de produtos
* Edição de produtos
* Exclusão de produtos
* Listagem de produtos
* Upload de imagens

## ☁️ Backend

* Integração com Supabase
* Banco de dados PostgreSQL
* Storage para imagens
* API REST
